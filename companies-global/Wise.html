海外公司

Wise（原 TransferWise）

一句话定位

来自爱沙尼亚、伦敦上市的全球低成本跨境汇款与多币种账户平台，以"P2P 撮合 + 透明费率"颠覆传统电汇市场，是 Open Banking 时代最具示范效应的"全球银行账户层"基础设施。

基本信息

  • 全称：Wise plc（原 TransferWise Ltd）
  • 创立时间：2011 年
  • 创始人：Kristo Käärmann、Taavet Hinrikus（两位爱沙尼亚人，曾分别在德勤、Skype 工作）
  • 总部：英国伦敦
  • 母公司/股东背景：早期由 Index Ventures、Andreessen Horowitz、Peter Thiel、Richard Branson 等投资；上市后多元化股东。
  • 是否上市：是（伦敦证券交易所主板，LSE: WISE，2021 年 7 月直接上市）
  • 最新市值：约 80~120 亿英镑区间（2024 年）
  • 最新员工规模：约 5,500 人（2024 年）

业务版图

  • Wise Send（个人跨境汇款）：核心业务，以中间汇率 + 透明 BPS 费率（通常 0.4%~0.7%）替代传统银行的"差价 + 隐形费"模式。
  • Wise Account / Wise Multi-currency Account：多币种账户，支持持有 40+ 货币、获取本地银行账号（USD ACH、EUR IBAN、GBP Sort Code、AUD BSB 等），是 Wise 最差异化的"全球银行账户"产品。
  • Wise Card：万事达 / Visa 多币种借记卡，具备实时汇率消费、自动换汇与 ATM 取款，是数字游民群体广泛持有的"事实国际卡"。
  • Wise Business：面向中小企业的跨境收付款 + 雇员薪酬 + 多账户管理，2024 年收入增速显著。
  • Wise Platform：作为基础设施的 API 与白标方案，被 Monzo、N26、Bank Mandiri、Standard Chartered、Interactive Brokers 等数十家银行和金融科技公司接入，对消费者隐身但 TPV 可观。
  • 直接接入支付清算系统：是英国 CHAPS、欧洲 SEPA 和澳洲 NPP、加拿大 IDP 等清算系统的非银直接成员，部分国家通过日本资金转移业者牌照与新加坡 MPI 牌照展业。

关键时间线

  • 2024-09：将欧元与英镑结算客户余额接入欧洲央行的"主账户"试点，是非银接入央行清算的代表性案例。
  • 2024-04：与 Standard Chartered（渣打）合作，上线"代理行替代方案"型跨境产品。
  • 2023-Q4：股价随英国零售大行 Banking-as-a-Service 概念升温而走强。
  • 2023-06：CEO Kristo Käärmann 因税务问题被英国 FCA 罚款 36.5 万英镑（个人罚款），并被列入 FCA "fitness" 关注名单。
  • 2022-02：宣布从"TransferWise"改名为 Wise，强调多币种账户战略。
  • 2021-07-07：在伦敦证券交易所完成 Direct Listing，是英国科技公司近年来最大规模直接上市之一，开盘市值约 88 亿英镑。
  • 2018-04：宣布成为英国央行 CHAPS 系统的非银直接参与者，是英国第一家获此身份的金融科技公司。
  • 2017-05：发布 Borderless Account（多币种账户）。
  • 2015-01：与 Number26（后来的 N26）等欧洲新型银行合作，将其汇款 API 嵌入合作方 App。
  • 2011-01：在伦敦由两位爱沙尼亚创始人成立，目标是打破传统银行电汇手续费模式。

商业模式与盈利结构

  • 主要收入构成：
  • 跨境换汇费（占比 70%+）：在中间汇率上加约 0.4%~0.7% 的透明费用；
  • Wise Card 与日常消费的 Interchange / 取款费；
  • Wise Platform B2B API 接入费；
  • 客户余额产生的利息收入（"Interest income"），高利率环境下成为重要利润来源。
  • Take Rate：跨境换汇 Take Rate 约 0.5%~0.7%，长期目标是降至 0.5% 以下；
  • 关键财务指标（2024 财年，截至 2024 年 3 月）：营收 11.45 亿英镑，跨境量 1,180 亿英镑，调整后 EBITDA 利润率约 22%，含利息收入的总收入 13.4 亿英镑。

监管处境

  • 主要监管者：英国 FCA、比利时 NBB（Wise Europe SA）、新加坡 MAS、美国各州 Money Transmitter、日本金融厅、澳大利亚 ASIC。
  • 持牌情况：在英国持有授权电子货币机构牌照（API），在比利时持有银行子公司、新加坡 MPI、美国全 50 州 Money Transmitter、日本资金转移业者等。
  • 关键监管事件：
  • 2023 年因 CEO 个人税务问题被 FCA 罚款。
  • 2022 年 NYDFS 对 Wise 美国子公司提出反洗钱合规改善要求。
  • 长期被部分国家本地银行游说要求加强反洗钱、KYC 标准。

客观评估

Wise 是过去十年最成功的"非银跨境清算层"，以低费率 + 透明定价对传统银行 SWIFT/电汇模式形成结构性冲击，并通过 Wise Account、Wise Card、Wise Platform 在跨境数字游民、SMB 与银行白标三个赛道形成正反馈。其优势在于产品口碑、直接接入清算系统的牌照壁垒、远高于同行的客户净推荐值；挑战在于：本质上仍是低 Take Rate 的"管道"业务，对规模与利率环境敏感；与 Revolut 在多币种账户和卡产品上的正面竞争加剧；CEO 个人合规事件对监管关系造成一定阴影；Wise Platform 是否能成为真正的银行替代基础设施还需要更多大型银行案例验证。

与中国对标

  • 在中国对标"个人跨境汇款"赛道时，更接近招商银行 / 中行的留学汇款 + 蚂蚁的 WorldFirst（已被蚂蚁收购）的合体，但 Wise 的去 SWIFT 化基础设施层在中国受外汇管制无法直接复刻。
  • Wise 多币种账户在中国对应"出海者使用的香港/新加坡虚拟账户 + 招行香港 + ZA Bank"组合，但中国大陆居民使用 Wise 受限于个人外汇额度与跨境合规。
  • Wise Platform B2B 接入逻辑接近蚂蚁 GCash/Dana 投资矩阵的另一种形式：Wise 不投资银行，而是作为基础设施嵌入到银行 App 内。
